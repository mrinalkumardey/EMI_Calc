# Your Easy EMI Calculator  
**Bank-Grade • 100% Accurate • Made for Real Indian Banking Rules**

Live Demo → https://github.com/mrinalkumardey/EMI_Calc  

A clean, responsive, and highly accurate EMI calculator built with pure HTML, CSS & JavaScript – no frameworks, no backend, instantly deployable.

### Features
- Bank-accurate EMI calculation with ceiling rounding (e.g., ₹1,00,000 @ 22.25% × 12 months = ₹9,382)
- Real Indian bank EMI date logic:
  - Disbursement on 28th–2nd → EMI on 10th
  - Disbursement on 4th–9th → EMI on 3rd
  - First EMI strictly between 7–31 days from disbursal
- Three clean switchable tabs:
  1. EMI Calculation
  2. Processing Fee + GST Calculator
  3. Full Amortization Schedule with Pie Chart
- Export amortization schedule as beautifully formatted PDF
- Fully responsive – works perfectly on mobile & desktop
- No dependencies except Chart.js & jsPDF (loaded via CDN)
- 100% offline capable after first load

### Tech Stack
- HTML5
- CSS3 (Vanilla – mobile-first & modern design)
- Vanilla JavaScript
- Chart.js (Pie chart – Principal vs Interest)
- jsPDF + autoTable (PDF export)

### How to Use
1. Clone or download this repo
2. Open `index.html` in any browser – works instantly!
3. Or deploy to GitHub Pages (already configured)

### Deploy on GitHub Pages (2 minutes)
1. Fork this repo
2. Go to Settings → Pages → Source → Deploy from `main` branch `/ (root)`
3. Your site will be live at: `https://yourusername.github.io/emi-calculator-web`

### Note
> Actual EMI and Amortization Schedule may vary slightly depending on the lender's internal policies (e.g., exact day-count convention, holiday adjustments, etc.).
